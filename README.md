# 🚀 Facilis.dev

### The AI-First Web Application Builder

**Facilis.dev** is a high-fidelity, agentic AI platform that allows you to build, preview, and synchronize web applications in real-time. By leveraging the power of modern **LLMs** (Gemini, Claude, OpenAI), it transforms natural language instructions into fully functional HTML/CSS/JS sandboxes, providing a seamless and highly interactive development experience.

---

## ✨ Key Features

### 🤖 AI-Powered Development (Multi-Provider Support)

- **Prompt-to-App**: Instantly generate complete web applications from simple text prompts.
- **Model Flexibility**: Full support for **Gemini**, [ OpenAI, Claude in the future ]
- **Iterative Refinement**: Chat with the AI to refine UI elements, add logic, or debug on the fly.
- **File Extraction**: The system automatically identifies, extracts, and persists code blocks generated by the AI into a virtual project filesystem.

### ⏳ Time Machine (Advanced Project Versioning)

- **Automatic Snapshots**: Every significant change triggered by the AI is automatically versioned (rotating history of 20 snapshots).
- **Instant Preview**: Browse your project's history in a dedicated sidebar. Click any version to see a live preview instantly without affecting your current work.
- **Linear Rollback & History Sync**: Restoring an old version performs a project-wide synchronization, including clearing future versions and cleaning up the **AI chat history across all threads** to ensure total contextual consistency.

### 🏗️ Real-Time Project Sandbox

- **Instant Live Preview**: Uses highly optimized Blob URLs and the `srcdoc` iframe API to render your code in real-time as the AI writes it.
- **Multi-device Layouts**: Test your application on Mobile, Tablet, and Desktop resolutions with one click.
- **Isolated Environment**: Your generated app runs in a secure sandbox, preventing interference with the main workspace.

### ☁️ Persistent Cloud Synchronization

- **Offline-First but with sync**: All project data, files, and chat history are stored locally for maximum responsiveness.
- **Live Sync**: Automatic, real-time synchronization with a backend instance via a secure proxy.
- **Sync Status Loader**: A professional synchronization overlay ensures all data is perfectly up-to-date before you start your session.

### 💬 Intelligent Multi-thread Chat

- **Organized Threads**: Create separate discussion threads for different features or sections of your project.
- **Smart Context**: For new threads, the system automatically feeds the current project state into the AI to ensure it understands the existing architecture.

## 🚀 Getting Started

### Prerequisites

- [PNPM](https://pnpm.io/) (Mandatory package manager)
- [Node.js](https://nodejs.org/) (LTS recommended)
- A running [CouchDB](https://couchdb.apache.org/) instance

### 1. Installation

```bash
git clone https://github.com/kele23/facilis-dev.git
cd facilis-dev
pnpm install
```

### 2. Configure Environment

Backend settings can be defined in `nitro.config.ts` or as environment variables:

```env
# Backend & Auth
NITRO_COUCH_URL=http://your-couchdb-instance:5984
NITRO_COUCH_SECRET=your-couch-auth-secret
NITRO_JWT_SECRET=your-secure-jwt-key
NITRO_ENCRYPTION_SECRET=your-data-encryption-key
```

> [!IMPORTANT]
> **AI Tokens** (OpenAI, Gemini, OpenRouter, etc.) are **not** stored as environment variables. They are configured **per project** directly within the Application Settings UI for maximum flexibility and security.

### 3. Launch Development

```bash
pnpm run dev
```

## 🏗️ Project Structure

- `/src/views`: Core pages (Home, Project View).
- `/src/components`: UI components (Sandbox, Version History, Thread List).
- `/src/composable`: Database management and live sync hooks (`useDB.ts`).
- `/server/api`: Secure proxy routes for AI and configuration.
- `/server/assets/prompts`: System instructions and behavior definitions for the assistant.
